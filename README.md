# Search Duplicate Files

This node.js script displays any duplicate files in the current directory and all sub-directories recursively.

***Run time: T(O) = O(4n)***

### Build on:
 - node.js
 - fs module
 - no dependancies

Enter "node searchDuplicateFiles.js" in the terminal to run.
